# subvenciones-app

Buscador de subvenciones públicas para empresas españolas. Devuelve, a partir de un NIF y un perfil mínimo, las subvenciones nacionales (BDNS) y europeas (Funding & Tenders) más relevantes con scoring y razonamiento.

**Estado:** en diseño. Implementación pendiente.
